# Expense Tracker / Gestor de Gastos

Simple expense tracker made in Python.  
Gestor de gastos simple hecho en Python.

## Features / Funciones
- Add expenses / Agregar gastos
- View total and detailed list / Ver total y lista detallada
- Input validation / Validación de entrada
- Interactive menu / Menú interactivo

## How to run / Cómo ejecutar
python gestor_gastos.py
